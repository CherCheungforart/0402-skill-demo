# 抖音 AI 创作 K-pop 变装转场技能集成方案

本方案指导如何将 `kpop-mv-outfit-transition` 技能的能力与当前 React 前端项目（`ios-interaction-page`）进行真实集成。

## 1. 架构评估：现状与可行性

**结论：完全可行，但需要引入后端或 Serverless 中间层。**

### 1.1 当前前端状态（纯 UI）
目前的 React 项目 (`Home.tsx`) 是一个纯前端的 UI 壳。它的“对话”、“打字机效果”和“视频展示”都是硬编码的静态逻辑。

### 1.2 目标架构（真实 AI 驱动）
要让这个界面真正拥有生成 K-pop 变装脚本和视频的能力，架构需要升级为：
`React 前端 (UI) <---> 后端 API (Node.js/Python) <---> AI 大模型 (LLM) / 视频生成 API`

## 2. 核心对接流程

### 第一步：将技能转化为 System Prompt
`kpop-mv-outfit-transition` 本质上是一个精心设计的 Prompt 工作流（包含 `SKILL.md`, `prompt-template.md`, `style-guide.md`）。

你需要在后端代码中，将这三个文件的内容合并，作为给大模型（如 GPT-4, Claude 3, 或字节跳动的大模型）的 **System Prompt（系统提示词）**。

*   **角色设定**：告诉模型它是一个“抖音 AI 创作的韩系 K-pop MV 变装转场专家”。
*   **规则注入**：将 `SKILL.md` 中的“执行流程（先图后字、三点校准、7段输出结构）”喂给模型。
*   **知识库注入**：将模板和 6 大子风格（A-F）作为上下文提供给模型。

### 第二步：前端与大模型 API 对接（文本生成阶段）

当用户在界面中点击“A. 甜美风格的变装”并上传图片后，前端需要发起请求：

1.  **用户输入组装**：
    前端将用户的选项转化为自然语言：“我要变装。风格选A甜美风格。请根据我的图片生成变装方案。”
2.  **图片处理**：
    如果模型支持多模态（视觉理解），将用户上传的 `vv3.jpg` 转为 Base64 或 URL 随请求一起发送给大模型，让大模型提取“主体画像”。
3.  **流式响应 (Streaming)**：
    为了实现完美的“打字机”体验（取代现在的 `setTimeout` 假动画），后端应使用 SSE (Server-Sent Events) 或 WebSocket，将大模型生成的 7 段文本（目标复述、三点校准...）逐字或逐块推送到 React 前端。前端监听数据流并实时渲染。

### 第三步：视频生成对接（多模态阶段）

大模型生成了完美的“第 5 部分：提示词包”后，下一步是真正生成视频，这需要对接视频生成模型（如 Sora, Runway, 或者是字节跳动内部的图生视频 API）。

1.  **提取 Prompt**：
    后端自动从大模型返回的文本中提取出“提示词包”的内容。
2.  **调用视频 API**：
    将提取出的提示词，连同用户最初上传的参考图 (`vv3.jpg`)，作为参数发送给视频生成 API。
3.  **状态轮询与视频展示**：
    视频生成通常需要几分钟。前端需要展示一个 Loading 状态（比如现在的三个跳动小圆点）。后端轮询视频生成状态。生成完毕后，将真实的视频 URL 返回给前端，替换掉写死的 `<video src="/outfit-transition-a.mp4">`。

## 3. 落地建议（在 Coze 上的极速实现方案）

如果你想最快地验证这个想法，而不是自己从头写后端和对接大模型 API，**强烈建议使用 Coze（扣子）**。

1.  **在 Coze 创建 Bot**：
    *   **人设与回复逻辑**：将 `SKILL.md`、模板和风格指南的内容全部复制进去。
    *   **插件（Plugins）**：添加“图生视频”相关的插件（Coze 内部通常集成了火山引擎的视频生成能力）。
2.  **工作流（Workflow）设计**：
    在 Coze 中画一个工作流：接收用户图片和选项 -> 大模型分析并生成分镜脚本和提示词 -> 将生成的提示词自动传递给“视频生成插件” -> 返回最终的视频。
3.  **前端集成（Coze Web SDK / API）**：
    Coze 提供了 API 和 Web SDK。你可以将当前的 React 页面中的写死逻辑，替换为调用 Coze Bot API。用户在 React 界面点击选项，实际上是向你的 Coze Bot 发送了一条消息，Bot 会返回真实的文本和视频。

这样，你既保留了自己设计的完美 iOS 交互 UI，又拥有了真实的 AI 变装生成能力！
